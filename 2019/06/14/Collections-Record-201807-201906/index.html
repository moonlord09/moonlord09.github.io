Collections Record #1

M.O.O.N
2019-06-14

最后修改时间：2019-09-10 22:40

注：若无特殊标明，所有唱片的记录格式为：歌手/演奏者 - 专辑/合集名 [Label] // 出版日期
注：若无特殊标明，所有漫画作品的记录格式为：作者 - 漫画名 [Label] // 出版日期
注：若无特殊标明，所有文学作品的记录格式为：作者 - 作品名称 [出版社] // 出版日期
注：若无特殊标明，所有游戏作品的记录格式为：开发者 - 游戏名称 // 发行日期

自从去年，也就是2018年开始接触唱片和黑胶之后，对这类东西突然产生一种难以言喻的感觉。小时候家里因为生意的缘故（爸妈年轻时开了个舞厅），就摆着几台音响，Mixer，CD机，莫名其妙地对物理介质的存储器有着天然的亲和感。
有一段时间我比较痴迷于数字资源的收藏，但是当去年第一次摸到黑胶，将这盘唱片毕恭毕敬地放在机子上播放时，我产生了一种别样的感觉。我想到《银翼杀手2049》里当K在向Wallace公司的数据库管理员询问问题时，那位光头的俏皮管理员说的一句话：

“那时我们几乎把所有的东西都储存在网络，服务器上，所有的东西。随着大断电，啪！全没了。”

经过差不多一整年的收集（雾），趁着毕业离校整理东西的空隙，完整地记录一下。当然也有很多没有收集到，也希望在今后的日子里，能一张张地将它们从世界各地搜集，好好地保存起来。美好的东西值得记录，不应该被储存在虚拟的网络世界里。（一个CS学生的危险发言

在日后一些整理的过程中，在文档中加入了别的一些收藏或者作品的记录，为避免篇幅过长导致无法正常的查阅，加入标题和标签以供方便的查找。
